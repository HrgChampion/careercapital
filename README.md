# Career Returns
This repository contains a small TypeScript financial engine in `lib/mbaEngine.ts`.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run a type-check:

```bash
npm run typecheck
```

3. Build the library to `dist/`:

```bash
npm run build
```

Notes
- The code is written in TypeScript and the `lib/` folder is included by `tsconfig.json`.
- After `npm install` you can run the `typecheck` script to validate types.
